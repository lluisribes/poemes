# Recopilació de versos i poemes | Collection of Verses and Poems

> *Si vis amari, ama*

## Català

Aquest repositori conté una **recopilació de versos i poemes** escrits durant els darrers anys. Moltes de les peces aquí recollides són fruit dels exercicis realitzats a les meves **classes d'escriptura** poètica.

### Objectiu i Format
L'objectiu d'aquest espai a GitHub és doble:
*   **Salvaguarda:** Servir com a sistema de preservació i còpia de seguretat de l'històric de textos publicats originalment a [www.lluisribes.net](https://www.lluisribes.net).
*   **Accessibilitat Tècnica:** Oferir el contingut en un format còmode i universal com és el **.md (Markdown)**. Aquesta elecció facilita el tractament dels textos mitjançant eines d'anàlisi de dades, gestors de coneixement personal i el processament a través de models de llenguatge (LLM).

---

## English

This repository is a **collection of verses and poems** written over the past years. Many of these pieces originated as exercises within my **writing classes**.

### Purpose and Format
The goal of this GitHub space is twofold:
*   **Safeguard:** To serve as a preservation and backup system for the historical archive of texts originally published at [www.lluisribes.net](https://www.lluisribes.net).
*   **Technical Accessibility:** To provide the content in a convenient and universal format: **.md (Markdown)**. This ensures the texts are optimized for data analysis tools, personal knowledge management systems, and processing by Large Language Models (LLMs).
